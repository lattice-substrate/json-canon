# Algorithms

**Status:** Draft

These documents specify the core algorithms for a strict JCS implementation:

1. **07-01-tokenizer.md** — RFC 8259 tokenizer and strict number lexing
2. **07-02-utf8-validation.md** — strict UTF‑8 validation rules (RFC 3629)
3. **07-03-string-unescape.md** — JSON escapes, surrogate pairs, I‑JSON forbidden code points
4. **07-04-object-sorting.md** — UTF‑16 code unit sorting and duplicate detection
5. **07-05-number-serialization.md** — ECMAScript-compatible number formatting approach
6. **07-06-emitter.md** — canonical JSON emission rules (whitespace, escaping, UTF‑8)

All normative claims are sourced to RFC 8785 / 7493 / 8259 / 3629.
