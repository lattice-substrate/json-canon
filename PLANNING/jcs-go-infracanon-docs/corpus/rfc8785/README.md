## number_samples.json
This file is a convenience set of numeric values around RFC 8785 boundary cases. It is **not** a golden-output vector.  
Golden number mappings are tracked in `corpus/numbers/rfc8785_appendix_b.csv` and in the V8 differential corpus.

Sources:
- RFC 8785 Appendix B (number samples): https://www.rfc-editor.org/rfc/rfc8785
- RFC 8785 V8 validation guidance: https://www.rfc-editor.org/rfc/rfc8785
